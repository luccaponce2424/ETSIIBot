# ETSIIBot
Un **chatbot** para **Discord** que asiste a los alumnos de la ETSII con humor y amabilidad.

## Comandos

| Comando | Descripción |
|---------|-------------|
| echo | Repite el mensaje enviado |
| help | Muestra una lista de comandos disponibles |
| info | Muestra información del bot, versión y dependencias con sus versiones |
| meme | Envía una imagen meme aleatoria desde Cloudinary |
| ping | Mide tiempo de respuesta generando un mensaje con IA |
| reset | Resetea la memoria del bot para el usuario que envía el comando |
| tokens | Muestra información sobre el uso de tokens de Cohere |

## Instalación

Para instalar las dependencias:
```
npm install
```

Para arrancar el bot:
```
npm start
```

Si además quieres que el bot se reinicie automáticamente cuando se detecten cambios (modo desarrollo), utiliza este comando para arrancarlo:
```
npm run dev
```
